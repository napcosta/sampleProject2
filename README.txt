
 I) O código disponibilizado nesta fase funciona, por omissão, com a implementação das tarefas nativas - pthreads.
 
II) Para o lançamento da aplicação servidor:
   1) Executa do comando "make" na directoria snfs+sthreads
   2) muda para a directoria snfs_server
   3) lançar na linha comandos ./server ( A mensagem "Server is running ...." indica que a aplicação já foi iniciada)     
   
II) Para a os programas que contêm os testes:  
   1) Abre um novo terminal
   2) Muda para a directoria onde se encontram os testes
   3) executa o "make"
   4) lança os testes com o servidor já iniciado noutro terminal.
